# Django E-Commerce Platform (Платформа для дистрибуции IT-курсов)

Веб-платформа (интернет-магазин) с отказоустойчивой асинхронной архитектурой, предназначенная для коммерческой дистрибуции цифрового контента и онлайн-курсов в сфере информационных технологий. 

## 🚀 Ключевой функционал и архитектура

- **Каталог и фильтрация:** Динамическое отображение курсов, гибкая система фильтров и кастомные шаблоны для представления контента.
- **Управление пользователями:** Расширенная модель профилей пользователей, кастомная система авторизации (модуль `authentication.py`), безопасная смена и сброс паролей через email-уведомления.
- **Асинхронная архитектура (Celery + Redis):** Вынос ресурсоемких процессов (генерация отчетов, транзакционные письма, фоновые логи) в распределенную очередь задач для обеспечения минимального времени отклика (TTFB) веб-интерфейса.
- **Безопасность и стандарты:** Разграничение прав доступа, защита от базовых уязвимостей, изоляция конфигурационных данных. Код полностью соответствует стандарту **PEP 8**.

## 🛠 Технологический стек

- **Backend:** Python 3.13, Django Web Framework 5.2+
- **База данных:** PostgreSQL / SQLite
- **Асинхронные задачи:** Celery, Redis (как брокер сообщений и кэш)
- **Фронтенд-часть:** HTML5, CSS3 (модульная структура стилей)
- **Инструменты разработки:** Git, python-dotenv (управление конфигурацией)
